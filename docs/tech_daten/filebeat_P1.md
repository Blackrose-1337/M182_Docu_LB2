# Filebeat

Filebeat ist ein leichtgewichtiger Log-Datenversender, der von Elastic entwickelt wurde. Es ist darauf ausgelegt, Log-Daten von verschiedenen Quellen zu sammeln, zu transformieren und an Elasticsearch oder Logstash zu senden. In diesem Kontext wird Filebeat konfiguriert, um Log-Daten von bestimmten Dateien zu sammeln, zu verarbeiten und an Elasticsearch zu senden.
